Upload Instructions

Upload structured cloud configuration data in JSON or YAML format. Good sources include exported provider settings, IaC templates, Terraform-style config snapshots, deployment manifests, or policy documents.

Supported: JSON Supported: YAML Sources: AWS / Azure / GCP
  • Include fields like public access, firewall rules, encryption, MFA, logging, backups, and password policy.
  • Export settings from cloud dashboards or use infrastructure-as-code files for the best scan accuracy.
  • Use the sample files if you need a quick demonstration dataset.
